# Aulas das disciplinas de Programação para Web - PW25S e PW44S - Turma 5SI

## Back-end 

A API REST será desenvolvida utilizando o *framework* **Spring**.

### Softwares
	- JDK 17 ou superior
	- IDE:
		- ItelliJ IDEA
		- Spring Tools Suite 4
		- Eclipe for JavaEE ...
	- SDBG:
		- Postgresql
	- Ferramenta para testar a API:
		- Postman
		- Insomnia
	- Git
	- Docker
